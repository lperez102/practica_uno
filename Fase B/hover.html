¿Qué es la media query HOVER?

La media query hover permite detectar si el dispositivo del usuario puede interactuar al pasar el cursor sobre un elemento. Esto sucede normalmente en computadoras con mouse o trackpad.

En muchos dispositivos móviles esta acción no existe, porque la interacción ocurre tocando directamente la pantalla. Por eso esta media query permite aplicar estilos solo cuando el dispositivo realmente soporta hover.

¿Cuándo se utiliza?

Se utiliza cuando queremos agregar efectos visuales que dependen del cursor. Por ejemplo:

  • Cambiar el color de un botón
  • Agrandar una tarjeta
  • Resaltar enlaces
  • Mostrar información adicional

Esto permite que la experiencia sea mejor en computadoras, sin depender de interacciones que no existen en pantallas táctiles.

Media query utilizada

@media (hover: hover)

Dentro de esta regla se usa también :hover para cambiar el estilo del elemento cuando el cursor pasa sobre él.

Conclusión

La media query hover ayuda a crear estilos más adecuados según el tipo de dispositivo. De esta forma, la interfaz puede aprovechar mejor las interacciones de escritorio sin depender de acciones que no existen en dispositivos táctiles.
