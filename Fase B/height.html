Media Query: Min Height

Exploración de la media query min-height y cómo permite adaptar el diseño según la altura de la pantalla.

¿Qué es la media query MIN-HEIGHT?

La media query min-height permite aplicar estilos cuando la altura de la pantalla es mayor o igual a un valor específico.

Esto significa que el diseño puede cambiar dependiendo del espacio vertical disponible en la pantalla del usuario.

¿Cuándo se utiliza?

Se utiliza cuando queremos aprovechar mejor pantallas con mayor altura. Por ejemplo:

  • Agrandar secciones en pantallas grandes
  • Aumentar el espacio entre elementos
  • Mostrar contenido adicional
  • Adaptar layouts para pantallas grandes

Media query utilizada

@media (min-height: 700px)

Dentro de esta regla se cambian los estilos cuando la altura de la pantalla es mayor o igual a 700px.

Conclusión

La media query min-height permite adaptar el diseño dependiendo de la altura de la pantalla, lo que ayuda a aprovechar mejor el espacio disponible en diferentes dispositivos.
