Media Query: prefers-reduced-motion

Exploración de la media query prefers-reduced-motion y cómo permite reducir animaciones para mejorar la accesibilidad.

¿Qué es PREFERS-REDUCED-MOTION?

La media query prefers-reduced-motion detecta si el usuario ha configurado su sistema para reducir las animaciones.

Algunas personas prefieren menos movimiento en las interfaces porque las animaciones pueden causar distracción, mareo o incomodidad visual. Esta media query permite adaptar el diseño para ofrecer una experiencia más accesible.

¿Cuándo se utiliza?

Se utiliza cuando un sitio web tiene animaciones o transiciones que pueden ser reducidas para mejorar la accesibilidad. Por ejemplo:

  • Desactivar animaciones largas
  • Reducir efectos de movimiento
  • Eliminar transiciones complejas
  • Crear interfaces más cómodas para algunos usuarios

Media query utilizada

Conclusión

La media query prefers-reduced-motion permite crear interfaces más accesibles adaptándose a las preferencias del usuario. De esta forma, los sitios web pueden ofrecer experiencias más cómodas para personas sensibles al movimiento.
