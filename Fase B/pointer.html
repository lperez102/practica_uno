¿Qué es la media query POINTER?

La media query pointer permite detectar qué tipo de dispositivo de entrada utiliza el usuario para interactuar con la página. Esto puede ser un mouse, un dedo en una pantalla táctil o incluso un stylus.

Dependiendo del dispositivo, la precisión del puntero puede cambiar. Por ejemplo, un mouse es muy preciso, mientras que el dedo en una pantalla táctil es menos preciso. Por esta razón, esta media query permite adaptar el diseño según el tipo de puntero disponible.

¿Cuándo se utiliza?

Se utiliza cuando queremos ajustar la interfaz dependiendo de la precisión del dispositivo de entrada. Por ejemplo:

  • Aumentar el tamaño de botones en pantallas táctiles
  • Separar más los elementos interactivos
  • Mejorar la accesibilidad para usuarios de dispositivos móviles
  • Optimizar la interacción para mouse o pantallas táctiles

Esto ayuda a que los elementos sean más fáciles de usar dependiendo del tipo de dispositivo con el que el usuario interactúa.

Ejemplo práctico

En este ejemplo, el botón cambia su tamaño cuando el dispositivo utiliza un puntero menos preciso, como una pantalla táctil.

Media query utilizada

@media (pointer: coarse)

Dentro de esta regla se pueden modificar estilos para dispositivos táctiles, donde la precisión del puntero es menor.

Conclusión

La media query pointer permite adaptar la interfaz según el tipo de dispositivo de entrada que utiliza el usuario. Esto ayuda a mejorar la usabilidad, especialmente en dispositivos táctiles donde los elementos necesitan ser más grandes y fáciles de presionar.
